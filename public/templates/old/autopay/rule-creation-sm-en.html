FAQs

Can I edit a rule later?

Payment rule names and descriptions can be edited, but their configuration cannot be changed. Instead, you'll need to archive a rule, then create and activate a replacement rule with your preferred configuration.

If I archive a rule what happens to my existing scheduled payments?

Any scheduled payments will not be affected. The archived rule will not be applied to future reservations.

What happens if a payment fails?

You will be notified via e-mail. You can also view all failed payments under Automated Payments tab.

Can a payment rule be applied to multiple rate plans?

No. Each payment rule can only apply to one rate plan, but can be applied to multiple channels.

What happens when a card fails validation?

Automated Payments won't be able to schedule payments if a supplied card fails validation. You'll need to request an alternative card and manage payments for that reservation manually.

What happens if no rule matches?

Reservations that don't match any rules need payments processed manually. Reasons include when a modification or cancellation is received, or previous payments have been processed.

Automated Payments currently supports a limited number of channels. You may still need to manage some payments manually using your existing process.
